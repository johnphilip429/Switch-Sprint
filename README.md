A minimal, focused daily tracker for your job switch journey. Built with React + Tailwind.

## Features

- **Daily Session Tracker**: Flexible timer for your daily study routine.
- **Job Application Pipeline**: Track applications, status, and follow-ups.
- **14-Day Study Plan**: Integrated roadmap for SQL, Python, and Spark.
- **Analytics**: Visualize your progress and consistency.
- **Resources**: Manage your learning bookmarks.
- **Local Storage**: Data persists in your browser. No login required.

## Getting Started

1.  Current directory:
    ```bash
    cd switch-sprint
    ```
2.  Install dependencies (if not already done):
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
- Date-fns
